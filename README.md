# 단비
사용자 주거환경(햇빛,습도,통풍)을 고려하여 식물을 추천해주는 웹사이트입니다.  

<b>< 주요 기능 ></b>
<p>1.식물 맞춤추천</p>
<p>2.게시판</p>
<p>3.식물 검색<br/></p>
